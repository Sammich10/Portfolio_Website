Bubble sort works by iterating through each item in the array, and testing if it is smaller or larger than the element next to it. If the left element is larger, then the items are swapped, thus the smaller items 'bubble' up to the top over each iteration of the array. The algorithm keeps track of weather a swap was made or not, and stops once it iterates throug the whole list without making a swap.



Example implementation in C++

Selection sort works by iterating through the list and 'selecting' the smallest value, and it places this value at the end of the sorted array.



Example implementation in C++

Insertion sort works by iterating through the unsorted list and sorting each item into it's correct ordered position by swapping the the element with the element to it's left if the right element is smaller than the left, and continues until the element is 'inserted' into the correct position.



Example implementation is C++
